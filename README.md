<h1 align="center">Macmagotchi</h1>

<p align="center">macOS 메뉴 막대에서 함께 지내는 픽셀아트 펫</p>

## 설명

Macmagotchi는 작업을 방해하지 않는 메뉴 막대형 가상 펫 앱입니다. 메뉴 막대의 픽셀 펫을 클릭해 상태를 확인하고, 밥 주기·놀아주기·재우기·쓰다듬기로 돌볼 수 있습니다. 펫은 화면 하단을 산책하며, 상태와 기분에 따라 표정이 달라집니다.

- **지원 환경:** macOS 14 이상
- **기술:** Swift 6, SwiftUI, AppKit
- **저장 위치:** `~/Library/Application Support/Macmagotchi/pet.json`

## 빌드 및 실행

### Xcode

1. `Macmagotchi.xcodeproj`를 Xcode에서 엽니다.
2. 실행 대상 **Macmagotchi**를 선택합니다.
3. <kbd>⌘R</kbd>로 빌드하고 실행합니다.

### Swift Package Manager

```bash
swift build
swift run
```

> 알림 권한은 앱 번들로 실행할 때만 요청됩니다. `swift run`은 UI를 빠르게 확인하는 용도로 사용하세요.

## 기능

### 펫 돌보기

| 기능 | 효과 |
| --- | --- |
| 밥 주기 | 배고픔을 채우고 기분·친밀도를 높입니다. |
| 놀아주기 | 기분과 친밀도가 오르며 체력을 사용합니다. |
| 재우기 | 체력을 회복합니다. |
| 쓰다듬기 | 기분과 친밀도를 높입니다. |

각 행동은 설정한 집중 타이머가 끝난 뒤 적용됩니다. 타이머 중에는 메뉴 막대에 남은 시간이 표시됩니다.

### 상태와 성장

- 배고픔, 기분, 체력, 친밀도 네 가지 상태를 관리합니다.
- 30분마다 배고픔과 체력이 감소하며, 앱을 다시 열면 지난 시간도 반영합니다.
- 친밀도에 따라 레벨과 성장 단계가 바뀝니다.
- 배고프거나 지치면 펫 표정이 달라지고, 상태가 매우 낮을 때 macOS 알림을 보냅니다.

### 픽셀 펫과 데스크톱 모드

- 고양이, 토끼, 곰 중에서 새 펫을 선택할 수 있습니다.
- 메뉴 막대 아이콘과 팝오버에서 픽셀 애니메이션을 표시합니다.
- 선택적으로 화면 하단의 Dock 옆 공간을 걷는 데스크톱 펫을 표시합니다.

### 설정

- 한국어·영어 전환
- Light·Dark·Pastel 테마
- 행동별 집중 시간 조절 (5–480분)
- 데스크톱 펫 표시 전환 및 펫 데이터 초기화
